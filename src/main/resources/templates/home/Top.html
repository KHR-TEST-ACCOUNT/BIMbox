Welcom to BIMbox!

"BIMbox" はシェアハウスでのコミュニティの活性化を目的とした掲示板型チャットサービスです。

About BIMbox

User Suggest

プロフィールに趣味を登録しておくと、コミュニティー内で共通点を持ってる人を探しその人物をサジェストします。

Search for a user

いつでも気になるユーザーの情報を検索する事ができ、自分と気が合いそうなユーザーを見つける事ができます。

Share idea, photo

アイデアや写真などをシェアする事ができ、コミュニティー内でのみ公開されます。（匿名投稿機能を追加予定です。）

1 on 1 chat

気になるユーザーとの１対１のチャットルームを作る事ができます。まずは趣味の合う人を見つけてメッセージを送ってみましょう。

Function Details

Profile

自身のプロフィールを編集・更新することができます。

郵便番号の入力時には住所を自動で補完し入力することができます。

Admin権限では全てのユーザーのプロフィールを編集・更新することができますが、それ以外の権限では自分のプロフィールのみ編集できます。

More info

User List

BIMboxに登録しているユーザーの一覧を表示する事ができます。現在は最大で８件まで表示できます。

検索欄では、全て空白の場合は全件検索を行い、入力値がある場合は「ユーザー名・趣味・自己紹介文」の３つのうちのいずれかで曖昧検索を行います。

ユーザーのアイコンをクリックすることによって、そのユーザーの詳細情報を確認する事ができます。

More info

Community

ユーザーが作成したトピックや投稿を閲覧する事ができます。

投稿をお気に入りして評価する機能もあり、今後はお気に入りにいれた投稿のみを表示する機能を実装する予定です。

投稿に対するコメントも可能で、コメントと同時に画像も送信することができます。

More info

Chat Room

チャットの記録のあるユーザーとそのチャット履歴を閲覧する事ができます。

チャットはリアルタイム通信に対応しており、相手からのメッセージをすぐに確認できます。

自身の送信したチャットメッセージはいつでも削除する事ができますが、削除した旨のメッセージが表示され、 １週間後（現在は３日後に設定）にシステムによって自動的に削除されます。

Pick Up Functions

ユーザーサジェスト機能

  プロフィール画面で趣味を登録しておくと、 BIMboxに登録しているユーザーで同じ趣味を持っているユーザーをシステムが斡旋し、 おすすめのユーザーとして一覧にサジェストします。
  現在は趣味のみが対象ですが、 今後は興味のFavoriteしたトピックや閲覧したユーザーによってもシステム側から提案できるユーザーを増やしていく予定です。

  サジェストされているユーザーの情報をタップするとそのユーザーのプロフィール画面に遷移することができます。
  また、そのユーザーと話をしてみたい場合は１対１のチャットルームを作成しチャットを送り合う事ができます。

機能へ

ユーザー情報詳細表示機能

  UserList画面では、ユーザーの画像をタップするとそのユーザーの詳細な情報を閲覧することができます。
  現在は「年齢・誕生日・メールアドレス・趣味・自己紹介文・ユーザー登録日時」の６つの情報が確認できます。

  ユーザー詳細からはプロフィールやChatルームへも遷移することが可能ですが、 通信なしでユーザーの情報を確認することができます。
  今後はユーザーのお気に入り機能も実装し、お気に入りのユーザーのみの表示にも対応していく予定です。

機能へ

新規投稿機能

  Community画面では、最下部のアコーディオンメニューから新規トピックの投稿が行えます。
  投稿の題名と投稿内容を入力し、自由にアイデアや個人的なつぶやきを投稿することができます。
  投稿欄の下にはファイルのアップロード欄があり、投稿に画像を追加することも可能です。

  画像のアップロードはドラッグアンドドロップに対応しており、簡単に投稿したい画像を選択することができます。
  また、投稿した内容やトピックが自身のものであればいつでも自由に削除や編集を実施することが可能です。
  今後は複数枚の画像の投稿にも対応していく予定です。

機能へ

チャットユーザー一覧機能

  ChatRoom画面では、チャット中のユーザーを一覧表示で見ることができます。
  現在チャットルームに入室中のユーザーは背景色が変わり、クリックやタップで通信無しでチャットルームに戻ることができます。 　

  なお、この画面はウィンドウの縮小時に現れ、スマホやタブレットなどにもレスポンシブにも対応しています。
  チャットルームに入室しているユーザーとのチャットはリアルタイムに通信を行っているので、 今後は新たなチャットが送信されてきた場合に、閲覧していないチャット数を画面表示できるよう改修していく予定です。
